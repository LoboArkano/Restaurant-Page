## Getting Started

To get a local copy up and running follow these simple example steps.
```
    git@github.com:LoboArkano/Restaurant-Page.git
```

### Prerequisites

- npm

- Nodejs

### Setup

Install dependencies with:

```
    npm install
```

Save a change with:

```
   npm run build
```
or
```
   npm run watch
```

### Usage

Open index.html on your browser
